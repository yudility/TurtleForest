# TurtleForest

## 🎮 게임 소개
**"바다거북숲", 가상현실 VR 탈출 게임**<br>
늦은 시간, 숲에서 조난당한 상황.<br>
안전하게 탈출하기 위해 필요한 아이템을 수집하고 HP를 관리하자<br>

## ℹ 정보
개발 기간: 2023.05.~2023.06<br>
개발환경: Unity 2021.3.20f1<br>
기기: 오큘러스 퀘스트 2<br>
